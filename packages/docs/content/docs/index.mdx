---
title: Math Library
description: 3D visualization library with reactive atoms
---

A library for creating reactive 3D mathematical visualizations.

```typescript
import { Scene3D, View3D, vec3 } from "mathlib";

const scene = new Scene3D();
const cam = scene.create("camera3d", {
  position: vec3(10, 10, 10),
  lookAt: vec3(0, 0, 0),
  zoom: 2,
});

const view = new View3D(scene, cam.id, document.getElementById("container"));
```
