MathLib is a library for creating interactive 3D mathematical visualizations with reactive state management. Built on Three.js, it provides a declarative API where your visualizations automatically update when underlying data changes.

## Quick Example

```typescript
import { Scene3D, View3D, vec3 } from "mathlib";

// Create a scene to hold all objects
const scene = new Scene3D();

// Create a reactive value
const t = scene.atom(0);

// Create a helix curve that grows as t increases
const helix = scene.create("parametricfunction3d", {
  f: (u) => vec3(u, Math.sin(u), Math.cos(u)),
  tStart: 0,
  tEnd: t,  // Reactive! Curve updates when t changes
  color: "dodgerblue",
  samples: scene.atom((get) => get(t) * 16), // Dynamic sample count
});

// Add a camera and render to the DOM
const camera = scene.create("camera3d", {
  position: vec3(10, 10, 10),
  lookAt: vec3(0, 0, 0),
});

const view = new View3D(scene, camera.id, document.getElementById("container"));

// Animate the curve
function animate() {
  t.set((prev) => prev + 0.05);
  requestAnimationFrame(animate);
}
animate();
```

## Key Features

- **Reactive by design**: Values are atoms that automatically trigger re-renders when updated
- **Declarative API**: Describe what you want, not how to render it
- **TypeScript-first**: Full type safety and autocomplete support
- **Three.js powered**: Hardware-accelerated 3D rendering
