## What is a Circle2D?

A `Circle2D` renders a circle from a center and a radius. It can draw the outline (the circle itself), the filled disk, or both, and with an angular span it draws an arc or a sector instead of the whole circle. Circles are useful for:

- Drawing a circle around a center, such as in circle geometry
- Marking a radius of influence or a neighborhood around a point
- Shading a disk as a filled region
- Drawing an arc, or a sector (a pizza slice), with `thetaStart` and `thetaEnd`

A `circle2d` whose `center` is bound to a draggable `point2d`, so dragging the handle moves the circle while the slider sets its `radius`.

## Basic Usage

Pass a `center` and a `radius`:

```typescript
scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 2,
  strokeColor: "dodgerblue",
});
```

By default a circle is drawn as an outline only. The fill is off (`opacity` is `0`) and the outline is on, since the curve is the common case.

## Outline and Fill

A circle has two parts you can style independently: the outline (stroke) and the disk (fill).

### Outline

`strokeColor`, `strokeOpacity`, and `strokeThickness` control the outline:

```typescript
scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 2,
  strokeColor: "white",
  strokeOpacity: 1,
  strokeThickness: 3,
});
```

Setting `strokeThickness` or `strokeOpacity` to `0` hides the outline.

`strokeDashed` draws the outline dashed, with the pattern derived from `strokeThickness` and a constant on-screen rhythm at any zoom, the same behavior as a dashed [line2d](/docs/2d-items/lines). With an angular span it dashes just the arc, which suits implied or auxiliary arcs:

```typescript
scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 2,
  strokeColor: "white",
  strokeDashed: true,
});
```

### Fill

`color` and `opacity` control the filled disk. Raise `opacity` above `0` to shade it:

```typescript
scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 2,
  color: "dodgerblue",
  opacity: 0.3,
  strokeColor: "dodgerblue",
  strokeOpacity: 1,
  strokeThickness: 2,
});
```

To draw a solid disk with no outline, raise `opacity` and leave the stroke off.

## Arcs and Sectors

`thetaStart` and `thetaEnd` set the angular span, in radians, measured counterclockwise from the positive x-axis. They default to a full turn (`0` to `2π`). Give them a smaller span and the circle becomes an arc: the **stroke draws just the arc curve**, and the **fill draws the sector** (the pizza slice from the center).

```typescript
// A 90° arc outline, from 3 o'clock up to 12 o'clock.
scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 2,
  thetaStart: 0,
  thetaEnd: Math.PI / 2,
  strokeColor: "dodgerblue",
});
```

Raise `opacity` to fill the sector instead of (or alongside) drawing the arc:

```typescript
// A shaded quarter-disk sector.
scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 2,
  thetaStart: 0,
  thetaEnd: Math.PI / 2,
  color: "gold",
  opacity: 0.35,
  strokeOpacity: 0,
});
```

For a marked angle between two arms, reach for the [angleMark2D](/docs/constructions/angle-mark-2d) construction, which wraps a circle2d arc and computes the angle for you.

## Examples

### Reactive Center and Radius

Drive the center and radius from atoms so the circle follows a draggable handle and a slider:

```typescript
const handle = scene.create("point2d", {
  coords: vec2(0, 0),
  color: "gold",
});

const radius = scene.atom(2);

scene.create("circle2d", {
  center: handle.coords,
  radius,
  strokeColor: "gold",
});
```

Drag the handle and the circle follows; write to `radius` and it resizes.

### Ellipses and Spirals

`circle2d` covers circles, arcs, and sectors. For an ellipse, a spiral, or any other non-circular curve, use a [parametric curve](/docs/2d-items/parametric), which lets you set both coordinates as functions of the parameter:

```typescript
scene.create("parametricfunction2d", {
  f: (t) => vec2(2 * Math.cos(t), Math.sin(t)), // an ellipse
  tStart: 0,
  tEnd: Math.PI * 2,
  color: "dodgerblue",
});
```

## Updating Properties

```typescript
const circle = scene.create("circle2d", {
  center: vec2(0, 0),
  radius: 1,
});

circle.center.set(vec2(2, 1));
circle.radius.set(3);
circle.opacity.set(0.3); // shade the disk
circle.strokeColor.set("seagreen");
```

## API Reference

### Options

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `center` | `Vec2` | `(0, 0)` | Center of the circle |
| `radius` | `number` | `1` | Radius of the circle |
| `thetaStart` | `number` | `0` | Start of the angular span, in radians |
| `thetaEnd` | `number` | `2π` | End of the angular span, in radians (a smaller span gives an arc / sector) |
| `color` | `string` | `"white"` | Fill color |
| `opacity` | `number` | `0` | Fill opacity (0 hides the disk) |
| `strokeColor` | `string` | `"white"` | Outline color |
| `strokeOpacity` | `number` | `1` | Outline opacity (0 hides the outline) |
| `strokeThickness` | `number` | `2` | Outline width |
| `strokeDashed` | `boolean` | `false` | Draw a dashed outline (pattern derived from `strokeThickness`) |
| `visible` | `boolean` | `true` | Whether the circle is shown |
| `pointerEvents` | `"auto" \| "none"` | `"auto"` | Whether the circle receives pointer events |
| `tags` | `string[]` | `[]` | Custom tags |

### Returned Item

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Unique identifier |
| `center` | `BoundAtom<Vec2>` | Center atom |
| `radius` | `BoundAtom<number>` | Radius atom |
| `thetaStart` | `BoundAtom<number>` | Span start atom (radians) |
| `thetaEnd` | `BoundAtom<number>` | Span end atom (radians) |
| `color` | `BoundAtom<string>` | Fill color atom |
| `opacity` | `BoundAtom<number>` | Fill opacity atom |
| `strokeColor` | `BoundAtom<string>` | Stroke color atom |
| `strokeOpacity` | `BoundAtom<number>` | Stroke opacity atom |
| `strokeThickness` | `BoundAtom<number>` | Stroke width atom |
| `strokeDashed` | `BoundAtom<boolean>` | Dashed outline atom |
| `visible` | `BoundAtom<boolean>` | Visibility atom |
| `pointerEvents` | `BoundAtom<string>` | Pointer events atom |
| `tags` | `BoundAtom<string[]>` | Tags atom |
