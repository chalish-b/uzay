## What is an Overlay2D?

An `Overlay2D` renders an HTML element (text or LaTeX) anchored to a position in the plane. As the camera pans or zooms, the overlay stays attached to its world position but is always drawn as flat HTML on top of the scene, at a constant pixel size. Overlays are useful for:

- Labeling points, vectors, or regions
- Displaying equations next to curves
- Showing dynamic values that update with the scene
- Adding annotations to a visualization

## Basic Usage

```typescript
scene.create("overlay2d", {
  position: vec2(1, 2),
  content: "Point A",
});
```

### LaTeX Labels

Set `format` to `"latex"` to render math using KaTeX:

<Callout type="warn">
  To use LaTeX overlays, you must import the KaTeX stylesheet in your application:
  ```typescript
  import "katex/dist/katex.min.css";
  ```
</Callout>

```typescript
scene.create("overlay2d", {
  position: vec2(0, 3),
  content: "\\int_a^b f(x)\\,dx",
  format: "latex",
});
```

## Positioning

### Anchor

The `anchor` property controls where the label sits relative to its world position. Think of the anchor as which corner or edge of the label is pinned to the point:

```typescript
scene.create("overlay2d", {
  position: vec2(0, 0),
  content: "Origin",
  anchor: "left", // Left edge of the label is at the position
});
```

Available anchors: `"center"`, `"top-left"`, `"top"`, `"top-right"`, `"left"`, `"right"`, `"bottom-left"`, `"bottom"`, `"bottom-right"`.

### Pixel Offset

Use `offset` to shift the label by a fixed number of pixels from its anchored position. This is useful for nudging labels away from the items they annotate:

```typescript
scene.create("overlay2d", {
  position: vec2(0, 0),
  content: "Shifted label",
  offset: vec2(10, -5), // 10px right, 5px up
});
```

## Styling

### CSS Class and Inline Styles

```typescript
scene.create("overlay2d", {
  position: vec2(0, 2),
  content: "Styled",
  className: "my-label",
  style: "color: red; font-size: 18px; font-weight: bold;",
});
```

### Visibility

Toggle visibility without removing the overlay:

```typescript
const label = scene.create("overlay2d", {
  position: vec2(0, 0),
  content: "Now you see me",
});

label.visible.set(false);
label.visible.set(true);
```

## Examples

### Updating Content

Display a live value that changes over time:

```typescript
const t = scene.atom(0);

scene.create("overlay2d", {
  position: vec2(0, 3),
  content: scene.atom((get) => `t = ${get(t).toFixed(2)}`),
});
```

### Following a Point

Attach a label to a moving point:

```typescript
const coords = scene.atom(vec2(1, 1));

scene.create("point2d", { coords, color: "gold" });

scene.create("overlay2d", {
  position: coords,
  content: "P",
  anchor: "bottom-left",
  offset: vec2(6, -6),
});
```

When the point moves, the label follows automatically.

### LaTeX with Reactive Values

```typescript
const angle = scene.atom(0);

scene.create("overlay2d", {
  position: vec2(2, 2),
  content: scene.atom((get) => `\\theta = ${get(angle).toFixed(1)}^\\circ`),
  format: "latex",
});
```

## API Reference

### Types

```ts
type OverlayAnchor =
  | "center"
  | "top-left"
  | "top"
  | "top-right"
  | "left"
  | "right"
  | "bottom-left"
  | "bottom"
  | "bottom-right";
```

### Options

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `position` | `Vec2` | `{ x: 0, y: 0 }` | World anchor position |
| `content` | `string` | `""` | Text or LaTeX content |
| `format` | `"text" \| "latex"` | `"text"` | Render as plain text or KaTeX |
| `offset` | `Vec2` | `{ x: 0, y: 0 }` | Pixel offset from the anchored position |
| `anchor` | `OverlayAnchor` | `"center"` | Which part of the label aligns to the position |
| `visible` | `boolean` | `true` | Whether the overlay is shown |
| `className` | `string` | `""` | CSS class name |
| `style` | `string` | `""` | Inline CSS styles |
| `pointerEvents` | `"auto" \| "none"` | `"none"` | Whether the overlay receives pointer events |
| `tags` | `string[]` | `[]` | Custom tags for grouping |

### Returned Item

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Unique identifier |
| `position` | `BoundAtom<Vec2>` | Position atom |
| `content` | `BoundAtom<string>` | Content atom |
| `format` | `BoundAtom<string>` | Format atom |
| `offset` | `BoundAtom<Vec2>` | Pixel offset atom |
| `anchor` | `BoundAtom<OverlayAnchor>` | Anchor atom |
| `visible` | `BoundAtom<boolean>` | Visibility atom |
| `className` | `BoundAtom<string>` | CSS class atom |
| `style` | `BoundAtom<string>` | Inline style atom |
| `pointerEvents` | `BoundAtom<string>` | Pointer events atom |
| `tags` | `BoundAtom<string[]>` | Tags atom |
