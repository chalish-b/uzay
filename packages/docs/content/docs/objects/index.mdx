Objects are the visual elements that make up your 3D scene. MathLib provides several types of objects for different purposes.

## Creating Objects

Use `scene.create(kind, options)` to add objects to a scene:

```typescript
const point = scene.create("point3d", {
  coords: vec3(1, 2, 3),
  color: "red",
  radius: 2,
});
```

The first argument is the object type, and the second is an options object where you configure the object's properties.

## Reactive Properties

Every property can be either a plain value or an atom:

```typescript
scene.create("point3d", { radius: 2 });

const r = scene.atom(2);
scene.create("point3d", { radius: r });

r.set(5);
```

You can also use derived atoms for computed properties:

```typescript
const t = scene.atom(0);

scene.create("point3d", {
  coords: scene.atom((get) => {
    const time = get(t);
    return vec3(Math.cos(time), Math.sin(time), 0);
  }),
});
```

## Available Objects

| Object | Description | Common Use Cases |
|--------|-------------|------------------|
| [Point3D](/docs/objects/points) | A sphere at a 3D position | Markers, vertices, handles |
| [Line3D](/docs/objects/lines) | A line segment between two points | Connections, edges |
| [Vector3D](/docs/objects/vectors) | An arrow with origin and direction | Forces, velocities, normals |
| [ParametricFunction3D](/docs/objects/parametric) | A curve defined by a parametric equation | Helixes, spirals, paths |
| [Sphere3D](/docs/objects/spheres) | A sphere with center and radius | Volumes, regions, atoms |
| [Overlay3D](/docs/objects/overlays) | A 2D label anchored to a 3D position | Labels, equations, annotations |
| [Axes3D](/docs/objects/axes) | Coordinate axes | Reference, orientation |
| [Grid3D](/docs/objects/grid) | A grid on a plane | Reference planes, floors |
| [Camera3D](/docs/objects/camera) | A viewpoint into the scene | Required for rendering |

## Removing Objects

Remove objects from the scene with `scene.remove()`:

```typescript
const point = scene.create("point3d", { coords: vec3(1, 1, 1) });

// Later
scene.remove(point);
```

## Tags

All objects support a `tags` property for categorization:

```typescript
const p1 = scene.create("point3d", {
  coords: vec3(0, 0, 0),
  tags: ["3d", "2d"],
});

const p2 = scene.create("point3d", {
  coords: vec3(1, 0, 0),
  tags: ["3d"],
});
```

Tags are currently unused. In the future, it'll be used for grouping objects, and different views and cameras will be able to filter items by tag.
