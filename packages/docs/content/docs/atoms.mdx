---
title: Atoms
description: Reactive state primitives
---

Atoms provide reactive state that automatically updates the visualization.

## Primitive Atoms

```typescript
const xAtom = scene.atom(0);

// Update the value
xAtom.set(5);

// Read the value
console.log(xAtom.get());

// Subscribe to changes
xAtom.sub(() => {
  console.log("xAtom changed:", xAtom.get());
});
```

## Derived Atoms

Create atoms that depend on other atoms:

```typescript
const position = scene.atom(vec3(1, 1, 1));
const scaled = scene.atom((get) => Vec3.scaled(get(position), 2));
```

## Connecting to DOM

```typescript
const slider = document.querySelector("#slider") as HTMLInputElement;
const valueAtom = scene.atom(0);

slider.addEventListener("input", (e) => {
  valueAtom.set(+(e.target as HTMLInputElement).value);
});
```
