## What is Vec3?

`Vec3` is a simple object type representing a 3D vector with `x`, `y`, and `z` components:

```typescript
type Vec3 = { x: number; y: number; z: number };
```

It's used throughout MathLib for positions, directions, and any 3D coordinate data.

## Creating Vectors

Use the `vec3()` helper function to create vectors. Notice the lowercase name and no `new` keyword.

```typescript
import { vec3 } from "mathlib";

const origin = vec3(0, 0, 0);
const position = vec3(1, 2, 3);
const direction = vec3(0, 1, 0); // Up
```

## Using with Atoms

Vectors work seamlessly with the atom system:

```typescript
// Static position
const point = scene.create("point3d", {
  coords: vec3(1, 2, 3),
});

// Reactive position
const posAtom = scene.atom(vec3(0, 0, 0));
const point = scene.create("point3d", {
  coords: posAtom,
});

// Update position
posAtom.set(vec3(5, 5, 5));

// Update based on the previous value
posAtom.set((prev) => Vec3.add(prev, vec3(1, 1, 1)));
```

## Vector Operations

The `Vec3` namespace provides common vector math operations. All operations return new vectors.

### Addition

```typescript
const a = vec3(1, 0, 0);
const b = vec3(0, 1, 0);
const c = vec3(0, 0, 1);

const sum = Vec3.add(a, b);     // { x: 1, y: 1, z: 0 }
const total = Vec3.add(a, b, c); // { x: 1, y: 1, z: 1 }
```

### Subtraction

```typescript
const a = vec3(5, 3, 1);
const b = vec3(1, 1, 1);

const diff = Vec3.subtract(a, b); // { x: 4, y: 2, z: 0 }
```

### Scaling

```typescript
const v = vec3(1, 2, 3);

const doubled = Vec3.scaled(v, 2);   // { x: 2, y: 4, z: 6 }
const halved = Vec3.scaled(v, 0.5);  // { x: 0.5, y: 1, z: 1.5 }
const negated = Vec3.scaled(v, -1);  // { x: -1, y: -2, z: -3 }
```

### Normalization

Get a unit vector (length = 1) pointing in the same direction:

```typescript
const v = vec3(3, 4, 0);
const unit = Vec3.normalized(v); // { x: 0.6, y: 0.8, z: 0 }
```

### Dot Product

```typescript
const a = vec3(1, 0, 0);
const b = vec3(0, 1, 0);

Vec3.dot(a, b); // 0 (perpendicular)
Vec3.dot(a, a); // 1 (parallel to itself)
```

### Cross Product

Get a vector perpendicular to both inputs:

```typescript
const x = vec3(1, 0, 0);
const y = vec3(0, 1, 0);

const z = Vec3.cross(x, y); // { x: 0, y: 0, z: 1 }
```

### Converting to Array

```typescript
const v = vec3(1, 2, 3);
const arr = Vec3.asArray(v); // [1, 2, 3]
```

## Constants

Commonly used vectors are available as frozen constants:

```typescript
Vec3.ZERO // { x: 0, y: 0, z: 0 }
Vec3.ONE  // { x: 1, y: 1, z: 1 }
```

These are frozen objects and cannot be modified.

## Example: Circular Motion

```typescript
const t = scene.atom(0);

const point = scene.create("point3d", {
  coords: scene.atom((get) => {
    const angle = get(t);
    return vec3(Math.cos(angle), Math.sin(angle), 0);
  }),
  color: "gold",
});

// Animate
function animate() {
  t.set(t.get() + 0.02);
  requestAnimationFrame(animate);
}
animate();
```

## API Reference

### Constructor

| Function | Signature | Description |
|----------|-----------|-------------|
| `vec3` | `(x: number, y: number, z: number) => Vec3` | Create a Vec3 object |

### Constants

| Constant | Value | Description |
|----------|-------|-------------|
| `Vec3.ZERO` | `{ x: 0, y: 0, z: 0 }` | Origin / zero vector (frozen) |
| `Vec3.ONE` | `{ x: 1, y: 1, z: 1 }` | Unit vector in all directions (frozen) |

### Operations

| Function | Signature | Description |
|----------|-----------|-------------|
| `Vec3.add` | `(...vectors: Vec3[]) => Vec3` | Sum of all input vectors |
| `Vec3.subtract` | `(a: Vec3, b: Vec3) => Vec3` | Vector subtraction `a - b` |
| `Vec3.scaled` | `(v: Vec3, scalar: number) => Vec3` | Multiply vector by scalar |
| `Vec3.normalized` | `(v: Vec3) => Vec3` | Unit vector in same direction |
| `Vec3.dot` | `(a: Vec3, b: Vec3) => number` | Dot product |
| `Vec3.cross` | `(a: Vec3, b: Vec3) => Vec3` | Cross product |
| `Vec3.asArray` | `(v: Vec3) => [number, number, number]` | Convert to tuple |
