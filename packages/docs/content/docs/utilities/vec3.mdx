<Callout type="warning">
These are lightweight utilities for basic vector math within Uzay. For a more complete, mature, and optimized solution, consider dedicated libraries like [gl-matrix](https://glmatrix.net/), [Three.js math](https://threejs.org/docs/#api/en/math/Vector3), or [mathjs](https://mathjs.org/).
</Callout>

`Vec3` is the 3D counterpart of [Vec2](/docs/utilities/vec2), used throughout Uzay for positions and directions: an immutable class with readonly `x`, `y`, `z` and chainable methods that always return a new vector.

Construct vectors with the `vec3` helper function, no `new` needed:

```typescript
import { vec3 } from "uzay";

const position = vec3(1, 2, 3);
const doubled = position.scale(2); // vec3(2, 4, 6)
```

A plain `{ x, y, z }` object is not a `Vec3`; adopt one explicitly when it arrives from outside:

```typescript
import { Vec3 } from "uzay";

const v = Vec3.from({ x: 1, y: 2, z: 3 });
```

## Operations

All operations return new vectors; nothing mutates.

### Arithmetic

```typescript
const a = vec3(1, 0, 0);
const b = vec3(0, 1, 0);

a.add(b);   // vec3(1, 1, 0)
a.sub(b);   // vec3(1, -1, 0)
a.scale(3); // vec3(3, 0, 0)
a.neg();    // vec3(-1, 0, 0)
```

### Length and Normalization

```typescript
const v = vec3(1, 2, 2);

v.len();   // 3
v.lenSq(); // 9 (faster, no sqrt)
v.unit();  // vec3(0.333..., 0.666..., 0.666...)
```

`unit()` on the zero vector returns the zero vector.

### Dot and Cross Product

```typescript
const x = vec3(1, 0, 0);
const y = vec3(0, 1, 0);

x.dot(y);   // 0 (perpendicular)
x.cross(y); // vec3(0, 0, 1), perpendicular to both
```

### Distance, Midpoint, Interpolation

```typescript
const a = vec3(0, 0, 0);
const b = vec3(4, 2, 6);

a.distTo(b);     // 7.48...
a.mid(b);        // vec3(2, 1, 3)
a.lerp(b, 0.25); // vec3(1, 0.5, 1.5)
```

`lerp` interpolates linearly from the receiver (at `t = 0`) to the argument (at `t = 1`); values outside `[0, 1]` extrapolate along the same line. `a.mid(b)` is `a.lerp(b, 0.5)`.

### Comparison and Conversion

```typescript
vec3(1, 2, 3).equals(vec3(1, 2, 3)); // true (with an epsilon tolerance)
vec3(1, 2, 3).toArray();             // [1, 2, 3]
```

## API Reference

### Constructors

| Function | Signature | Description |
|----------|-----------|-------------|
| `vec3` | `(x: number, y: number, z: number) => Vec3` | The everyday constructor |
| `Vec3.from` | `(p: { x: number; y: number; z: number }) => Vec3` | Adopt a plain object |

### Constants

| Constant | Value | Description |
|----------|-------|-------------|
| `Vec3.ZERO` | `vec3(0, 0, 0)` | Origin / zero vector (frozen) |
| `Vec3.ONE` | `vec3(1, 1, 1)` | Unit in all components (frozen) |

### Methods

All methods returning `Vec3` return a new vector.

| Method | Signature | Description |
|--------|-----------|-------------|
| `add` | `(v: Vec3) => Vec3` | Vector sum |
| `sub` | `(v: Vec3) => Vec3` | Vector difference (receiver minus argument) |
| `scale` | `(s: number) => Vec3` | Multiply by a scalar |
| `neg` | `() => Vec3` | Negation |
| `unit` | `() => Vec3` | Unit vector in the same direction (zero stays zero) |
| `len` | `() => number` | Magnitude |
| `lenSq` | `() => number` | Squared magnitude (faster) |
| `dot` | `(v: Vec3) => number` | Dot product |
| `cross` | `(v: Vec3) => Vec3` | Cross product |
| `distTo` | `(v: Vec3) => number` | Euclidean distance |
| `mid` | `(v: Vec3) => Vec3` | Point halfway to `v` |
| `lerp` | `(v: Vec3, t: number) => Vec3` | Linear interpolation toward `v` by `t` |
| `equals` | `(v: Vec3, epsilon?: number) => boolean` | Component-wise comparison with tolerance |
| `toArray` | `() => [number, number, number]` | Convert to tuple |
